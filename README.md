# Sync Monitor 🔄 for SillyTavern

**Sync Monitor** 是一个专为 SillyTavern (酒馆) 设计的轻量级扩展插件。它能实时监控浏览器与服务器之间的数据同步状态，防止因网络延迟或服务器写入未完成而导致的数据回档（配置丢失）。

## ✨ 功能特点

* **🛡️ 防回档监控**：实时检测配置保存、消息发送等后台请求。
    * 🟢 **绿色 (就绪)**：所有数据已同步，可以安全刷新。
    * 🟠 **橙色 (忙碌)**：正在写入数据，请勿刷新页面！
* **👆 全平台交互**：
    * **拖拽**：支持鼠标（电脑）和手指（手机）任意拖拽位置。
    * **记忆**：自动保存你放置的位置，刷新不重置。
* **🌗 双模式切换**：
    * **完整模式**：显示状态文字。
    * **极简模式**：单击即可缩成一个呼吸灯小圆点，不挡视线。

## 📦 如何安装

1.  复制本仓库的链接：
```
https://github.com/wominIII/Sync-Monitor
```
2.  打开 SillyTavern 酒馆。
3.  点击顶部的 **扩展 (Extensions)** 图标。
4.  点击 **安装扩展 (Install Extension)**。
5.  在 **Install from URL** 输入框中粘贴上面的链接。
6.  点击 **Install**，安装完成后刷新页面即可。

## 🎮 使用说明

1.  **移动位置**：按住状态条（或小圆点）即可拖动到屏幕任意位置。
2.  **切换模式**：单击状态条，可在「完整长条」和「极简圆点」之间切换。
3.  **状态指示**：
    * 看见 **橙色闪烁** 🟠 时，**千万不要刷新网页**。
    * 变为 **青色常亮** 🟢 后，代表数据已安全写入。
